# sudoku solver
This small program can solve a sudoku game.
it does the following
  - solve the sudoku
  - validate the sudoku
  - generate random sudoku that can be solved 
  
## How to run it?
  - make sure you have Java(TM) SE Runtime Environment (build 15.0.1+9-18) installed
  - dowload sudoku.jar
  - left click it
